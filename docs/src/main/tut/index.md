## Installation

Cats-Practice is available on scala 2.11, 2.12, 2.13 and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "com.rmehri01" %% "Cats-Practice-core" % "0.0.1")
```
